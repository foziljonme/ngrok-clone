Get Started

Why QuickTunel?

🚀 Fast setup

Start sharing in seconds. Just install and run a single command.

🔒 Secure tunnels

Each tunnel is authenticated and encrypted end-to-end with JWT tokens.

⚙️ Developer-friendly

Built with Node.js, WebSockets, and Express — fully open source.

Quick Start


# Install the client
npm install -g QuickTunel

# Start a tunnel
QuickTunel start --port 5173

# View your public URL
https://myapp.QuickTunel.dev
